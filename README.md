# 🌍 City Mate - Travel Assistance Chatbot

## 📌 About
This project is a travel assistance web application that helps users with guidance and feedback using a chatbot system.

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript
- Node.js
- MySQL

## ✨ Features
- Chatbot-based travel guidance
- Feedback system with emoji selection
- Simple and user-friendly interface
